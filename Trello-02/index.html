Página Web

Sobre a página web

Uma página web é simplesmente um documento que é renderizado ("mostrado") por um navegador. Tais documentos são escritos com uso da linguagem HTML (a qual iremos ver em maiores detalhes em outros artigos).

Principais elementos de uma página web

  • Design responsivo
  • Identidade visual
  • Navegação intuitiva
